# Slither.io Addon for Kodi

A simple Kodi addon that launches Slither.io in Brave browser in fullscreen/kiosk mode.

## Requirements

- Kodi (Matrix or later - Python 3.0.0+)
- Brave Browser installed at `/usr/bin/brave-browser`

## Installation

1. Download the addon as a zip file
2. In Kodi, go to **Add-ons** → **Install from zip file**
3. Select the downloaded zip
4. The addon will appear under **Programs**

## Usage

1. Open Kodi
2. Navigate to **Add-ons** → **Programs**
3. Select **Slither.io (Brave)**
4. The game will launch in Brave in fullscreen mode

## Features

- Launches Slither.io in Brave browser
- Fullscreen/kiosk mode for better gaming experience
- Error notification if Brave is not found
